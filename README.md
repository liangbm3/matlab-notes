# MATLAB 笔记

📚 中山大学电子与信息工程学院《MATLAB 计算与仿真》课程相关笔记。（其实差不多就是将教材复制粘贴了一遍）

## 📖 简介

本仓库为 笔者在学习 MATLAB课程时的相关笔记和知识整理，涵盖从基础语法到高级特性的完整内容。适用于：
- 想要快速了解并上手 MATLAB 的初学者
- 中大电信院选修了《MATLAB 计算与仿真》课程的学弟学妹
- 期末开卷考试的参考资料
